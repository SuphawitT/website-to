ชิ้นส่วนเครื่องยนต์สองจังหวะ


เครื่องยนต์ 2 จังหวะ ประกอบด้วยชิ้นส่วนต่างๆ ดังต่อไปนี้


1.ช่องดูดส่วนผสมอากาศและเชื้อเพลิง : ส่วนผสมอากาศและเชื้อเพลิงจะถูกดูดเข้าสู่ห้องข้อเหวี่ยงผ่านช่องดูด ลิ้นอากาศ ที่ติดตั้งอยู่ภายในช่องดูดทำหน้าที่เป็น วาล์วกันกลับ ( วาล์วทางเดียว ) เพื่อควบคุมการไหลของส่วนผสมอากาศและเชื้อเพลิง

2.พอร์ตถ่ายโอน : ส่วนผสมเชื้อเพลิงอากาศอัดจะถูกถ่ายโอนจากห้องข้อเหวี่ยงไปยังพื้นที่การเผาไหม้ผ่านพอร์ตถ่ายโอน

3.พอร์ตก๊าซไอเสีย : ก๊าซไอเสียจะถูกปล่อยออกจากพื้นที่การเผาไหม้ผ่านทางพอร์ตก๊าซไอเสีย

4.ห้องข้อเหวี่ยง : เป็นที่บรรจุชิ้นส่วนภายในของเครื่องยนต์ ส่วนผสมของอากาศและเชื้อเพลิง จะถูกอัดภายในห้องข้อเหวี่ยงก่อนที่จะเข้าสู่ ช่องถ่ายโอน

5.ลูกสูบ : เคลื่อนที่ เป็นเส้นตรง ระหว่าง จุดศูนย์ตายบน (TDC) และ จุดศูนย์ตายล่าง (BDC) ฟิล์มน้ำมันบางๆ ระหว่าง แหวนลูกสูบ และปลอกสูบจะแยกพื้นที่เผาไหม้ออกจากห้องข้อเหวี่ยง

6.หัวเทียน : ใช้เพื่อจุดระเบิดส่วนผสมของอากาศและเชื้อเพลิง เครื่องยนต์เบนซิน/เบนซินใช้หัวเทียนและเรียกว่า เครื่องยนต์จุดระเบิดด้วยประกายไฟเครื่องยนต์

7.เพลาข้อเหวี่ย : เพลาที่ใช้แปลง การเคลื่อนที่ เชิงเส้น ของลูกสูบให้เป็นการ เคลื่อนที่แบบหมุน

8.ก้านสูบ : เชื่อมต่อลูกสูบกับเพลาข้อเหวี่ยง โดยเฉพาะอย่างยิ่ง ไม่มีการเชื่อมต่อระหว่างก้านสูบและเพลาข้อเหวี่ยง เนื่องจาก ตลับลูกปืนโลหะธรรมดา และ น้ำมันหล่อลื่น แยกชิ้นส่วนทั้งสองออกจากกัน
